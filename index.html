榨菜肉丝汤

发表于 2025-11-08 分类于 cooking

榨菜肉丝汤

很简单，也记录一下

食材准备

  1. 一包榨菜 肉丝(肉丸也可) 鸡毛菜 两枚鸡蛋
  2. 盐 香油

步骤

  1. 水烧开，把肉丝放进去煮得差不多熟，把鸡蛋打进去，再放榨菜和鸡毛菜
  2. 都熟了尝一口汤，根据咸淡放盐
  3. 出锅加香油

蒸河蟹

发表于 2025-11-08 分类于 cooking

蒸河蟹

大闸蟹如何蒸

食材准备

  1. 螃蟹若干只
  2. 料酒（多一点最好一斤以上），盐，花椒，葱，姜

步骤

  1. 螃蟹扔锅里，倒入一斤酒和少许水，酒可以少一点但是味道可能没那么好，多放盐，放适量花椒
  2. 大火蒸13分钟左右，这个根据火力判断，建议十分钟左右，掰一条腿试一下

大蒜炒牛肚

发表于 2025-11-08 分类于 cooking

大蒜炒牛肚

买新鲜牛肚，不是卤过的

食材准备

  1. 牛肚丝
  2. 大蒜叶
  3. 小葱 姜 蒜 豆瓣酱 小米辣 糖 盐 料酒

步骤

  1. 牛肚丝放碗里生抽+料酒腌制
  2. 大蒜叶切段，蒜白切斜一点，不然太厚不容易熟
  3. 油热下蒜瓣，生姜，加入牛肚丝翻炒，加入一勺豆瓣酱。
  4. 再加入蒜叶继续炒，中间放少量糖，盐根据目前口味控制
