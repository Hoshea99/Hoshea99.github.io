  3. vim常用操作

    1. G 翻到最下面
    2. gg翻到最上面
    3. dd删除该行
    4. :10 跳到第十行
    5. /error 向下查找”error“
    6. ?error 向上查找“error”
    7. ctrl b 往回翻页 ctrl f 往后翻页
    8. u 撤销一次最近改动
    9. o在当前行下方新开一行，并进入插入模式
    10. x 删除当前光标字符
    11. yy 复制当前行（等效于y1y y3y就是复制本行到下面共三行，配合p使用复制）
    12. p粘贴到下一行
    13. dG从当前行删除到文件末尾
    14. 0 移动到当前行首 ^移动到当前行有数据的那个字符
    15. $移动到当前行尾

tmux

使用mutex

  • 新建

    tmux new -s work

  • 离开

    Ctrl + b 然后按 d (Detach)

  • 查看对话

    tmux ls

  • 重连对话

    tmux a -t work

  • 关闭对话

    tmux里输入exit

  • 分屏

    水平分屏（左右）：Ctrl + b 然后按 %

    切换到左侧窗格（如果没在左边）：Ctrl + b 然后按 左方向键

    垂直分屏（上下）：Ctrl + b 然后按 "

    切换到右侧窗格：Ctrl + b 然后按 右方向键
