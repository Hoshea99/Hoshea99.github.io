与C++的不同之处

由于我之前一直写的是C++，所以我从与C++的对比开始进行学习。

运行机制

  • python是解释型，代码由解释器一行一行地翻译成机器码并执行。翻译过程会耗费时间。
  • C++是编译型，代码在运行前必须通过编译器整体转换成机器码（二进制文件）。

内存管理

  • python有自动垃圾回收机制
  • C++需要手动管理内存

类型

  • python是动态强类型，变量不需要声明类型

  • C++需要声明类型

在 Python 中，每一个 .py 文件在运行时，系统都会偷偷给它定义一个内置变量，叫 __name__。这个变量的值取决于你是怎么运行这个文件的：

  • 场景 A：直接运行 如果你直接在终端输入 python myscript.py，那么在这个脚本里，__name__ 的值就会被自动设为 "__main__"。
  • 场景 B：被当作模块导入 如果你在另一个文件里写了 import myscript，那么在 myscript.py 内部，__name__ 的值就会变成它的文件名（即 "myscript"）。
