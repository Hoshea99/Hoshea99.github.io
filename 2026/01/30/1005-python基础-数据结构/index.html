1005-python基础-数据结构

发表于 2026-01-30 更新于 2026-03-05 分类于 python

python的数据结构和c++相比有所不同，这里学习一下具体用法。

列表

列表，List，可以有序地容纳不同类型的数据。

核心操作

操作 Python C++ (std::vector)
末尾添加 l.append(x) v.push_back(x)
弹出末尾 l.pop() v.pop_back()
合并容器 l.extend(other_list) v.insert(v.end(), other.begin(), other.end())
查找索引 l.index(x) std::find(v.begin(), v.end(), x)
清空 l.clear() v.clear()

这里要注意append和extend的区别

字典

核心操作

操作 代码示例 备注
创建 d = {"name": "张三"} 使用大括号 {} 或 dict()
新增/修改 d["age"] = 25 Key 不存在则新增，存在则覆盖
删除 (带返回值) val = d.pop("name") 删掉并返回该值，类似 C++ 的 extract
删除 (无返回) del d["age"] C++ 里的 erase
判断 Key 是否存在 "name" in d 返回布尔值，替代 C++ 的 m.count(k)
清空 d.clear() 彻底抹除所有键值对

集合

集合(set)，其实就是一个只有 Key、没有 Value 的字典。也正因如此，它查找元素的速度是 O(1)，且自带“排他性”。

核心操作

操作 代码示例 对应 C++
创建 s = {1, 2, 3} std::unordered_set<int> s {1, 2, 3};
添加 s.add(4) s.insert(4);
删除 s.remove(2) s.erase(2); (不存在会报 KeyError)
安全删除 s.discard(9) s.erase(9); (不存在不报错)
是否存在 x in s s.count(x) 或 s.find(x)
