linux新版常用指令

发表于 2026-02-28 更新于 2026-03-05 分类于 linux

工作环境中常用的指令再复习一下

文件和进程管理

文件操作

ls -al :看权限

cp -r: 递归复制

mkdir -p：递归创建目录

文件处理

grep "ERROR" app.log :过滤出带ERROR的行

awk '{print $2, $4}' app.log: 打印第二列和第四列

sed -i 's/debug=True/debug=False/g' config.py : 修改配置文件中的设置

权限管理

chown -R pi:pi /home/pi/my_project:递归改变路径下的所有者

chmod 755 script.py: 修改读写执行权限

进程监控

top

ps -ef |grep python

kill -9
